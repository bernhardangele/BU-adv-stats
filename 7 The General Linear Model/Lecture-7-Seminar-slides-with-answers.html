Understanding contrasts a bit more

  • For this seminar, we will use an invented data set that will let you experience a bit more about how contrasts work
  • Please download the following data set from Brightspace and open it in jamovi/SPSS: fluffy.csv
    • For SPSS, you may have to use the CSV import wizard (see here for detailed instructions)
    • jamovi will open the file immediately without issues.
  • These (fictitious) data are from an experiment in which participant rate the positivity of emotions triggered by looking at photos of different dogs and cats (on a scale from 0 - 100, where 0 is most negative and 100 is most positive).
  • An unrelated group of participants rated the same photos in terms of how fluffy they thought the animal was (also on a scale from 0 to 100, where 0 is sandpaper and 100 is extra-soft plush fabric).
  • Also, the file notes whether the photo showed a cat or a dog.

Your task

  • First, perform a regression analysis where you predict positive feelings only by fluffiness. Note down the results, and write one sentence to summarise what they suggest.

  • Then, perform a regression analysis where, in addition to fluffiness, you also add animal type (cat vs.dog) as a discrete predictor. How is this model different from the first one? How can you explain the difference?

Results: adding the contrast (in words)

  • In words: We fitted a linear model (estimated using OLS) to predict positive_feelings with fluffiness and animal (formula: positive_feelings ~ fluffiness + animal). The model explains a statistically significant and substantial proportion of variance (R2 = 0.84, F(2, 97) = 250.41, p < .001, adj. R2 = 0.83). The models intercept, corresponding to fluffiness = 0 and animal = cat, is at -42.83 (95% CI [-56.25, -29.41], t(97) = -6.33, p < .001). Within this model:

  • The effect of fluffiness is statistically significant and positive (beta = 1.13, 95% CI [0.94, 1.32], t(97) = 11.59, p < .001; Std. beta = 1.41, 95% CI [1.17, 1.65])

  • The effect of animal [dog] is statistically significant and positive (beta = 84.02, 95% CI [74.41, 93.63], t(97) = 17.35, p < .001; Std. beta = 4.20, 95% CI [3.72, 4.68])

Question

  • Why was the slope for fluffiness negative when only fluffiness was in the model, but positive when we have both fluffiness and animal in the model?
  • How is this related to the idea of statistically controlling for a variable?
  • Hint: plotting might help think of Anscombes Quartet!

Answer

  • Consider the plot of the analysis of fluffiness alone:

Answer (2)

  • And now the plot for the analysis including animal:

Answer (3)

  • As you can see, the line fitted when animal is not included is technically the best possible fit if you dont know that there are two groups, but in practice it ignores the trend we care about
  • When we take animal into account, we can fit the line within the groups instead of between
  • In this way, we can say that we have controlled for the effect of animal statistically we have removed the confound by including the confounding variable in the model
  • Of course, it does not always work this nicely in reality!
  • For example, the slope does not have to be the same for cats and dogs.

  • The value of the coefficient always tells us how much the y-value changes when the x-value changes by 1
  • When we use the 0 and 1 contrasts, that means the means for cat and dog are exactly 1 unit on the x-axis apart
    • That means 1 unit on the x-axis corresponds to 84 units on the y-axis
  • When we use the -1 and 1 contrasts, that means the means for cat and dog are exactly 2 unit on the x-axis apart
    • That means 1 unit on the x-axis corresponds to 42 units on the y-axis
